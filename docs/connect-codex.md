# Connect From Codex

## 연결 정보

- 서버 이름 예시: `Songsim Campus MCP`
- Streamable HTTP URL: `https://your-public-mcp-url/mcp`

## 주 사용 시나리오

- 성심교정 장소와 건물 검색
- 개설과목 조회
- 최신 공지 조회
- 출발 장소 기준 주변 식당 조회
- 지하철/버스 교통 안내 조회

## 예시 요청

- `성심교정 중앙도서관 정보 찾아줘`
- `2026년 1학기 컴퓨터정보공학부 과목 검색해줘`
- `최신 academic 공지 보여줘`
- `중앙도서관 근처 한식집 찾아줘`

## 공개 서버 제한

- 이 공개 MCP는 read-only입니다.
- profile, timetable, notice preferences, meal personalization, admin 기능은 제외됩니다.

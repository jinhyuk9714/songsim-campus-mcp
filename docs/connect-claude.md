# Connect From Claude

## 연결 정보

- 이름 예시: `Songsim Campus MCP`
- Streamable HTTP URL: `https://your-public-mcp-url/mcp`
- 용도: 성심교정 장소, 과목, 공지, 식당, 교통 안내 조회

## 추천 프롬프트

- `성심교정 건물 중 중앙도서관 정보 보여줘`
- `객체지향 과목 찾아줘`
- `지금 볼 만한 최신 공지 정리해줘`
- `중앙도서관 근처 카페 추천해줘`

## 공개 서버 제한

- read-only 조회 전용
- 개인화 profile과 admin/sync 기능은 로컬 full 모드에서만 사용
